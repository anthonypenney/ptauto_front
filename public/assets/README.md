# Asset Intake

Put source assets here so they can be reviewed and incorporated into the site.

## Folders

- `logos/` - logo files, wordmarks, alternate logo versions
- `photos/` - shop exterior, bays, vehicles, team, equipment, local business photos
- `favicon/` - favicon or icon source files

## Preferred Files

- Logos: SVG first, then PNG or high-quality JPG
- Photos: original JPG or PNG files
- Favicon: SVG, PNG, or ICO

Keep original filenames if they are meaningful. If not, names like `shop-exterior.jpg`, `front-desk.jpg`, `logo-main.png`, and `favicon-source.png` are ideal.
